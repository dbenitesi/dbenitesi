Introduction

Azure es una plataforma de informatica en la nube con un conjunto de servicios que se amplia continuamente

  • Almacenamiento Remoto
  • Hospedaje de bases de Datos
  • Administracion centralizada de cuentas
  • Inteligencia Artificial
  • Internet de las cosas(IoT)

Aspectos Basicos de Azure

El modulo de aspectos basicos de Azure es una serie de 6 rutas de aprendizaje.

¿Qué es la informática en la nube?

Es la entrega de servicios informaticos a traves de Internet.Estos servicios incluyen

  • Almacenamiento
  • Bases de Datos
  • redes
  • Software
  • Analisis e Inteligencia

Lo que nos ofrece: innovacion mas rapida, recursos flexibles y economias de escala
